About Project

Customer Purchase Prediction is a Machine Learning based project that predicts whether a customer is likely to purchase a product or not. The prediction is based on important customer attributes such as Age, Income, Experience and Customer Score. This project is developed using:


  • Python
  • Pandas
  • NumPy
  • Seaborn
  • Matplotlib
  • Scikit-Learn
  • Flask
  • HTML & CSS
